=====
Обзор
=====

Приложение QuickAPI - это простой путь установки механизма вызовов
для проектов Django.

.. note::

    Протестировано на Django 1.4 и 1.8.

.. warning::

    Мы настоятельно рекомендуем использовать защищённый протокол **HTTPS**!
